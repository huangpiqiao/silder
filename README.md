###组件功能
```
1、实现无缝滑动
2、实现组件自适应布局，以设计稿的宽度和网站主要的内容宽度为准
3、实现初始化自动滚动时下一张图片必须加载出来或者加载失败才会滚动
4、实现图片自动最大化并居中功能
```
